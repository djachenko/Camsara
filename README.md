# Camsara

Camsara (**cam**era + san**sara**, потому что с цветовыми колесами) — приложение-фотоассистент для анализа цветовых схем в кадре. 

Статус — пока в разработке.

## Описание

Приложение анализирует поток данных с камеры, считает распределение цветов в кадре, выбирает из них преобладающие и показывает результат на цветовом колесе. Пользователь (фотограф) может выбрать текущее фокусное расстояние, соответствующее объективу на отдельной камере, чтобы получать результаты под свой будущий кадр.

## Стек

Swift 5.9 без Concurrency, MVVM, SwiftUI + UIKit для поддержки iOS до 17 версии, AVFoundation, Combine, DITranquillity.
